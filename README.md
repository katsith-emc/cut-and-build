# Cut & Build — Sculpture Archive

木製の板材を切断し、切断面で再接合する彫刻シリーズ「Cut & Build」の記録リポジトリです。
均一な素材(15×9×1.5cmの白色塗装ボード)から、切断角度の違いによって非一様な結果を生成する過程を扱っています。

An artwork series visualizing the E=mc Thought Principle — uniform wooden boards (15×9×1.5cm, white-painted) cut once and rejoined at the cut face, generating non-uniform structural outcomes from uniform material.
